# 微商城后台管理系统

基于 Vue 3 + Element Plus + Vite 的现代化后台管理系统。

## 功能特色

- Vue 3 + Composition API
- Element Plus 组件库  
- ECharts 数据可视化
- Pinia 状态管理
- 响应式设计

## 主要功能

- 🏠 首页Dashboard - 数据统计和图表展示
- 📋 分类管理 - 树形分类结构管理
- 🛍️ 商品管理 - 商品信息增删改查
- 👤 个人中心 - 用户信息和密码管理

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
